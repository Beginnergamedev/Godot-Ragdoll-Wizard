# Godot-Ragdoll-Wizard
A tool that helps simplify the creation of ragdolls in godot with a custom node and a custom dock in the editor. Just add a ragdoll builder node and go to the ragdoll wizard dock at the bottom and just copy paste bone names into their correct slots and adjust physics settings, after that you can press "create ragdoll" and a physicalbonesimulator along with physicalbone nodes with optimized settings (e.g. joint constraints) and correctly scaled colliders are created. But you still need Jolt physics engine for this. This is very similar to unity's ragdoll wizard
